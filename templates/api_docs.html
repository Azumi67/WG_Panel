Access

Use your panel session in the browser. For scripts, send Authorization: Bearer <PANEL_API_KEY>. Node-agent examples use <NODE_API_KEY>.

Commands

Each API route includes a copy-ready command. Page routes show the browser URL because they are meant for the panel UI.

Testing

Start with health checks and list routes. Try create or update actions on a test interface before changing real peers.
